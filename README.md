# AI Executive Dashboard Generator

## 🔍 Problem
Business teams struggle to extract insights from messy CRM data.

## 💡 Solution

## ⚙️ Features
- Data cleaning and normalization
- KPI generation (Revenue, Win Rate, Pipeline)
- Interactive dashboards
- AI-style business insights
